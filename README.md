# react_mobx
Testing out Mobx and React

**Start by these two commands:**
1. **_yarn_**
2. **_yarn start_**

**Build and run on local server:**
1. **_yarn build_**
2. **_yarn add serve global_**
3. **_cd build_**
4. **_serve -s_**

_N.B. The full list of users can be checkout at **src/mocsk/api.js**_
